# Install and Create Environment

Navigate to /Intellihub/src/backend and run these commands

For first time running:
### py -3 -m venv .venv
### .venv/Scripts/activate
### pip install flask
### pip install pymongo
### pip install google-api-python-client
### pip install python-dotenv
### npm run start-api

Every time after: 
### .venv/Scripts/activate
### npm run start-api
